# jsInnerHTML

Research: Project4Dimensions

## Why jsInnerHTML?

jsInnerHTML is a JavaScript algorithm that can be used to insert new 
content or replace existing nodes of an HTML element.

See [`inner-html.js`](inner-html.js).
